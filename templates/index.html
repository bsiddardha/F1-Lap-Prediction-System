⚙️ Process

How It Works

Four steps from raw telemetry data to a predicted lap time.

01
📡
Data Collection

Historical F1 race telemetry from 2018–2024 seasons is collected including lap times, tyre data, car positions, and weather conditions.

02
🔧
Feature Engineering

Raw data is processed into meaningful features: tyre degradation curves, driver consistency scores, constructor performance indices.

03
🌲
Random Forest Model

A Random Forest Regressor is trained on thousands of race laps, learning complex non-linear relationships between inputs and lap times.

04
⚡
Real-Time Prediction

Enter your parameters and get an instant lap time prediction served via a FastAPI backend with sub-100ms response time.

🏁 About the Project

ML Meets Formula 1

This project combines a passion for Formula 1 racing with machine learning engineering. The model was trained on 7+ years of official F1 race data, capturing the nuances of each driver's style, team strategies, and tyre behavior.

Built as an end-to-end ML pipeline — from data scraping to model serving — it demonstrates real-world application of Random Forest models in a sports analytics context.
